## Install

```sh
npm install
```

## ENV configure

Create .env file and setup

```sh
VITE_SERVER_URL = your_graphql_server_url
```

## Usage

```sh
npm run dev
```

## Generate types for GraphQL

Configure codegen.yaml

```sh
schema: your_graphql_server_url
```

And you can run

```sh
npm run generate-types
```
